# Interactive Calculator

Портфолио-проект с несколькими интерактивными калькуляторами, сохранением
результатов, формой заявки с валидацией и имитацией отправки на backend.

## Живая версия

[https://cherreshenka1.github.io/interactive-calculator/](https://cherreshenka1.github.io/interactive-calculator/)

## Что есть

- Калькулятор стоимости сайта
- Калькулятор доставки
- Калькулятор ремонта
- Реал-тайм пересчёт
- Сохранение расчётов в localStorage
- Форма заявки с валидацией
- Имитация backend submit
- Анимации результата

## Запуск

```bash
npm install
npm run dev
```

## Сборка

```bash
npm run build
```

## Деплой

```bash
npm run deploy
```
